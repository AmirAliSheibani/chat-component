# Chat Component (React + Tailwind)

یک کامپوننت ساده چت با React و Tailwind CSS.  
کاربران می‌توانند پیام بفرستند و پاسخ‌های mock از سیستم دریافت کنند.

## ویژگی‌ها
- باکس ورودی پیام
- دکمه ارسال
- نمایش پیام‌های کاربر و پاسخ سیستم
- طراحی تمیز و ساده با Tailwind CSS
- راست‌چین مناسب برای فارسی

## پیش‌نیازها
- Node.js 18+  
- npm
## نحوه اجرا
1. پروژه را کلون کنید:
```bash
git clone https://github.com/USERNAME/REPO_NAME.git
cd REPO_NAME
```

2. نصب وابستگی‌ها:
```bash
npm install
```


3. اجرای پروژه:
```bash
npm run dev
```
4. باز کردن در مرورگر:
```bash
http://localhost:5173/
```

## فایل‌ها
```bash
src/components/Chat.jsx — کامپوننت چت
src/app.jsx — ریشه برنامه
src/index.css — فایل CSS با Tailwind
```
